# Install
- https://devdocs.bagisto.com/2.2/introduction/installation.html
- cp .env.example .env
- ensure you have actual website address in .env
- composer install
- php artisan bagisto:install

- npm add vite
- npm run build

- sudo chmod -R 777 app/storage
- sudo chmod -R 777 app/bootstrap/cache
